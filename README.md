# Каталог стран мира

Семестровая работа по JavaScript. Веб-приложение для просмотра информации о странах мира с использованием публичного API.

## Функциональность

- Приветственный экран с вводом имени пользователя
- Сохранение имени
- Загрузка данных о странах из внешнего API
- Отображение стран в виде карточек с флагом и информацией
- Сортировка по 6 направлениям (название, население, площадь — возр./убыв.)
- Фильтрация по региону, поиску и независимости
- Панель статистики (всего стран, среднее население, средняя площадь)
- Кнопка выхода из аккаунта

## Технологии

- HTML5
- CSS3
- JavaScript 
- REST Countries API
